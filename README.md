## Quick Start

```bash
# Clone the repository
git clone https://github.com/neverinfamous/container-manager.git
cd container-manager

# Install dependencies
npm install

# Run development build
npm run dev
```

## Development

```bash
# Build
npm run build

# Lint
npm run lint

# Type check
npm run typecheck

# Run tests
npm test

# All checks
npm run check
```
